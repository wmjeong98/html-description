# `<caption>`
- 테이블 제목
# `<col>`
- 테이블 열
- `span` : 연속 열의 수 지정 기본값은 1
# `<colgroup>`
- 테이블 열을 묶는 그룹 정의
# `<table>`
- 테이블
# `<tbody>`
- 테이블 본문
# `<td>`
- 데이터 포함하는 테이블의 셀 정의
- `colspan` : 열 합치기
- `rowspan` : 행 합치기
# `<tfoot>`
- 테이블의 열을 요약하는 행들의 집합
# `<th>`
- 테이블 셀 그룹의 헤더셀
- `abbr` : 셀 내용의 짧은 요약
- `align` : 정렬
- `colspan rowspan`
- `scope` : 테이블 헤더 셀의 범위 지정
# `<tr>`
- 테이블 셀 행 정의
# `<thead>`
- 테이블 열의 헤드를 정의하는 일련의 행을 정의
